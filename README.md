# Clock Tools
Sistema javascript para pegar o tempo entre uma data e outra repetidamente

Uso: clockTools.tableGenerator(*hora inicial*, *hora final*, *minutos em contador*);


Exemplo:
clockTools.tableGenerator('10:00', '12:00', 30);
-----------------
Retorna:
[ 10,0  ]
[ 10,30 ]
[ 11,00 ]
[ 11,30 ]
[ 12,00 ]
